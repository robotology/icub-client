Description

Module based on kinectClient "Kinect Wrapper Client" that tracks people. This module receives information from the Kinect Wrapper, and convert joints to the iCub referential. It requires the kinectServer, OPC and RFH running.

Parameters

  • –name: To specify the module's name; all the open ports will be tagged with the prefix /name
  • –from: To specify the configuration file
  • –context: To specify the default context where config files are loader
  • –period: period of acquisition in seconds
  • –opc: To specify the specifies the opc database name, where data are readed
  • –showImages: to have GUI with depth, images, joints, ...
  • –showMode: to select which images to show, as a sum of available types: rgb, depth, skeleton, players
  • –x: x position of the images as a group
  • –y: y position of the images as a group
  • –x-rgb: x position of the rgb image
  • –y-rgb: y position of the rgb image
  • –x-depth: x position of the depth image
  • –y-depth: y position of the depth image
  • –x-skeleton: x position of the skeleton image
  • –y-skeleton: y position of the skeleton image
  • –x-players: x position of the players image
  • –y-players: y position of the players image

Input Ports

  • /agentDetector/rpc [Bottle] [default carrier:tcp]: Response port. Supported commands:
    • change_partner_name NAME_STRING: change the name of partner
    • pause: lock with mutex
    • resume: unlock the mutex

Output Ports

  • /agentDetector/skeleton:o [yarp::sig::Bottle] [default carrier:tcp]: buffered port streaming out skeleton properties
  • /agentDetector/kinect/depthPort:o [yarp::sig::ImageOf] [default carrier:udp]: buffered port streaming out depth images
  • /agentDetector/kinect/imagePort:o [yarp::sig::ImageOf] [default carrier:udp]: buffered port streaming out images
  • /agentDetector/kinect/playerPort:o [yarp::sig::ImageOf] [default carrier:udp]: buffered port streaming out partner images
  • /agentDetector/kinect/skeletonPort:o [yarp::sig::ImageOf] [default carrier:udp]: buffered port streaming out skeleton images
  • /agentDetector/kinect/agentLoc:o [yarp::sig::Bottle] [default carrier:tcp]: buffered port streaming out agent location
  • /agentDetector/rfh:o [rpc] [default carrier:tcp]: port to communicate with referenceFrameHandler rpc server
