Detailed Description

Module to detect touch activations to robot's body parts Version:1.0

Author
Bertrand Higy bertr.nosp@m.and..nosp@m.higy@.nosp@m.iit..nosp@m.it
Copyright
Released under the terms of the GNU GPL v2.0

Description

This module provides wrappers of procedures to count the number of activations to robot 's body parts through some yarp ports.

Parameters

  • – name : name of the module

Input Ports

  • /touchDetector/torso:i [Bottle] [default carrier:udp]: Get touch activation from torso
  • /touchDetector/left_arm:i [Bottle] [default carrier:udp]: Get touch activation from left arm
  • /touchDetector/right_arm:i [Bottle] [default carrier:udp]: Get touch activation from right arm
  • /touchDetector/left_forearm:i [Bottle] [default carrier:udp]: Get touch activation from left forearm
  • /touchDetector/right_forearm:i [Bottle] [default carrier:udp]: Get touch activation from right forearm
  • /touchDetector/left_hand:i [Bottle] [default carrier:udp]: Get touch activation from left hand
  • /touchDetector/right_hand:i [Bottle] [default carrier:udp]: Get touch activation from right hand

Output Ports

  • /touchDetector/touch:o [Bottle] [default carrier:udp]: Stream out the number of activations
  • /touchDetector/touchClean:o [Bottle] [default carrier:udp]: Stream out the touched taxel ID
