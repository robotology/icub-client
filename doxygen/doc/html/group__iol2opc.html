Detailed Description

Bridge module between the interactive object learning and recognition system (iol) and yarp database of object properties Version:1.0

Description

This module provides wrappers of procedures to acquire perceived information from iol and store them in opc database.

Parameters

  • – name : name of the module

Input Ports

  • /iol2opc/blobs:i [Bottle] [default carrier:udp]: Get extracted blobs from lbpExtract.
  • /iol2opc/histObjLocation:i [Bottle] [default carrier:tcp]: Receives the pixel in the image from which retrieve a cartesian position for displaying the classification histograms of the closest blob.
  • /iol2opc/img:i [Bottle] [default carrier:tcp]: Get calibrated images from camera.
  • /iol2opc/getClick:i [Bottle] [default carrier:tcp]: Get clicked position through this port

Output Ports

  • /iol2opc/imgTrack:o [Image] [default carrier:udp]: Stream out the image of tracked objects
  • /iol2opc/imgLoc:o [Image] [default carrier:udp]: Streams out the images for real-time objects localization.
  • /iol2opc/imgHistogram:o [Image] [default carrier:udp]: Stream out the histogram of classification scores.
  • /iol2opc/imgClassifier:o [Image] [default carrier:tcp]: Pass images to the classifier.
  • /iol2opc/imgSel:o [Image] [default carrier:tcp]: Stream out the image inside the selected blob.
  • /iol2opc/get3d:rpc [rpc] [default carrier:tcp]: Send requests to SFM for 3D estimation.
  • /iol2opc/classify:rpc [rpc] [default carrier:tcp]: Send requests to himrepClassifier for object classification.

Services

  • /iol2opc/rpc [rpc-server]: Command port . This service is described in iol2opc_IDL (iol2opc.thrift)
